## Project 1: Azure Manufacturing Data Pipeline

Engineered an automated cloud ETL pipeline in Microsoft Azure to process and warehouse 10,000 equipment sensor telemetry records. Raw manufacturing logs—covering process temperatures, rotational speed, torque, tool wear, and failure categories—are preprocessed using Python, staged in Azure Blob Storage, transformed via Azure Data Factory Mapping Data Flows, and loaded into Azure SQL Database for operational analytics.
