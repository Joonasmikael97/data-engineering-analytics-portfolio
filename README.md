### [Azure Manufacturing Data Pipeline](./01-azure-cloud-pipeline)

Engineered an cloud ETL pipeline in Microsoft Azure to process and warehouse 10,000 equipment sensor telemetry records. Raw manufacturing logs covering process temperatures, rotational speed, torque, tool wear, and failure categories are preprocessed using Python, staged in Azure Blob Storage, transformed via Azure Data Factory Mapping Data Flows, and loaded into Azure SQL Database for operational analytics.

### [E-Commerce & Customer Analytics](./02-ecommerce-customer-analytics)

Built an interactive Business Intelligence solution using Excel, Power BI, Power Query, M Language, and DAX to analyze over 540,000 multi-region transactions across 38 countries. Cleansed and transformed raw transactional records using Power Query with custom M scripts, built a star-schema data model, and engineered advanced DAX measures to analyze customer purchasing behavior, regional sales performance, retention cohorts, and Customer Lifetime Value (CLV).
